Format my WKRPT

  • No cover page needed
  • No page numberig needed
  • No image numbering (just write /label before the image)
  • No table numbering (just write /label before the table)
  • No title formatig (just put a star {**} before a heading, two stars {**} before heading 2, etc)
  • No need to make references (put the link where you find the info in square brackets in text [www.link.com]

Once you get the file back, open it and add your name and project name to the cover page!




Good luck!
